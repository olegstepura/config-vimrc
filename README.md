# config-vimrc
My personal vimrc config used on ArchLinux.

## Install:
### Debian:
```bash
curl https://raw.githubusercontent.com/olegstepura/config-vimrc/master/vimrc.local -o /etc/vimrc.local
echo "source /etc/vimrc.local" >> /etc/vimrc
```

### Ubuntu:
```bash
curl https://raw.githubusercontent.com/olegstepura/config-vimrc/master/vimrc.local -o /etc/vim/vimrc.local
echo "source /etc/vimrc.local" >> /etc/vim/vimrc
```
